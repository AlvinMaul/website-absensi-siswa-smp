# 📊 Sistem Absensi Siswa SMP Berbasis Web

## 📌 Deskripsi

Project ini merupakan aplikasi sistem absensi siswa berbasis web yang dibangun menggunakan PHP dan MySQL. Sistem ini dirancang untuk membantu sekolah dalam mengelola data siswa serta pencatatan kehadiran secara digital, sehingga lebih efisien dan terstruktur dibandingkan metode manual.

---

## 🚀 Fitur Utama

* ✅ Manajemen Data Siswa (Create, Read, Update, Delete)
* ✅ Pencatatan Absensi Harian
* ✅ Dashboard Statistik Absensi
* ✅ Integrasi Database MySQL
* ✅ Tampilan Responsif (jika pakai Bootstrap)

---

## 🛠️ Teknologi yang Digunakan

* **Backend**: PHP Native
* **Database**: MySQL
* **Tools**: XAMPP / phpMyAdmin
* **Frontend**: HTML, CSS, (Bootstrap jika ada)

---

## ⚙️ Cara Menjalankan Project

1. Install XAMPP
2. Jalankan Apache dan MySQL
3. Import database:

   * Buka phpMyAdmin
   * Buat database dengan nama `absensi_smp`
   * Import file `database/absensi_smp.sql`
4. Pindahkan folder project ke:

   ```
   C:\xampp\htdocs\
   ```
5. Jalankan di browser:

   ```
   http://localhost/website-absensi-siswa-smp
   ```
